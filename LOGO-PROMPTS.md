# 🎨 PixelEuro – Logo-Prompts (auf Viralität getrimmt)

Ziel: ein Logo, das **sofort erkennbar**, **als winziges Icon lesbar** und **teilbar/merkbar** ist.
Was viral funktioniert: **simpel + ein Maskottchen oder ein starkes Symbol + kräftige Farben**.

**Beste Tools (Stand 2026):**
- **Ideogram** → bestes Tool für **lesbaren Text** im Logo (Wortmarke „PixelEuro").
- **Midjourney v6 / Flux** → beste Bildqualität für Symbol & Maskottchen.
- **DALL·E / ChatGPT** → schnell, gut für Varianten & transparenten Hintergrund.

> Tipp: Erst 4–6 Varianten generieren, dann die beste in **Vektor** umwandeln
> (recraft.ai oder vectorizer.ai) → scharf in jeder Größe + als Favicon nutzbar.

## 🎯 So wählst du das virale Gewinner-Logo
1. Generiere von **Konzept 1 (Maskottchen)** und **Konzept 2 (Symbol)** je 6 Varianten.
2. **Schrumpf-Test:** Sieht es als 32×32-Favicon noch gut aus? Wenn ja → Top-Kandidat.
3. **3-Sekunden-Test:** Zeig es jemandem 3 Sek. – kann er es danach grob nachzeichnen? Dann ist es merkbar.
4. **Maskottchen = Content-Maschine:** Der Charakter kann in jedem TikTok/Reel auftauchen
   („PixiEuro zeigt dir die Wand") → Wiedererkennung = Reichweite.

### Farbpalette (passt zur Website)
- Primär-Blau: `#2f6bff`
- Cyan-Akzent: `#00d4ff`
- Dunkler Hintergrund: `#0b1020`
- Weiß/Text: `#eaf0ff`

### Favicon erstellen
Bestes Symbol als 512×512 PNG exportieren → favicon.io oder realfavicongenerator.net →
die Dateien nach `public/` legen und in `index.html` einbinden.
